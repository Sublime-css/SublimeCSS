Documentation:

How to set up SublimeCSS:

You can either click the "Download" link at the top of the page to download SublimeCSS, or you can simply link to it in the head of your document with the link shown at the top of the page. If you do choose to download it, then unzip the download and move the SublimeCSS folder into your current project.

How does SublimeCSS work?

To use SublimeCSS, simply apply the appropriate class to an element, and SublimeCSS will work it's magic!

What classes are available?

Here is a compleate list of classes for V0.1. If you downloaded an older version, please head over to the "README.md" file supplied with your download. Note: SublimeCSS is forward compatible, so all older versions will always be supported, but may have less capability.

Grid system:

sublime-gridRow creates a row for grids that can contain grid elements.

sublime-grid-*-15 creates a column, where * is a number from 1 to 15.

For example, sublime-grid-3-15 is a grid column that is 3/15ths the width of the screen.

Note: all grids must be placed inside a sublime-gridRow div to function. Grids are designed to work with Div elements.

Create responsive grids using media querys: for example, these documentation boxes are 5-15 on screens above md(medium), and 15-15 on smaller screens. The classes used are "sublime-grid-5-15 sublime-grid-md-15-15".

Button system:

Buttons are very simple in SublimeCSS. Simply create an anchor elememt and apply the apropriate class. The options are:

  • sublime-greyBtn -- a grey button
  • sublime-yesBtn -- an affirmitive button
  • sublime-noBtn -- a negitive button
  • sublime-warningBtn -- an alarming button
